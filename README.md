# Approximation Theory — Course code and data (TU)

This repository contains all code and data necessary to reproduce the figures and numerical results presented in the lecture notes for the Approximation Theory course at TU.

Examples:
- `Figure_8_1.m` → produces Figure 8.1 in the notes

(Exact filenames reflect the labels used in the lecture notes.)

## Requirements
- MATLAB R2021a or newer
- Chebfun (for Chebyshev / spectral computations)
